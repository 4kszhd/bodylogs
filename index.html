How to set up Supabase (free)
  1. Go to supabase.com → New project (free tier)
  2. Project created → go to Settings → API
  3. Copy Project URL and anon public key
  4. Go to Table Editor → New Table: name it bodylog
  5. Add these columns:
    user_id (text, not null)
    date (text, not null)
    data (jsonb)
    updated_at (timestamptz, default: now())
  6. In Authentication → Policies: enable RLS and add a policy allowing all operations (for personal use)
  7. Paste URL + key here → done! ✓
Tip: Use the same user ID on all your devices (phone, tablet, computer) to sync data.
